<h3 align="center">Extra Large Language Model Operating Systems</h3>
<p align="center"><i>From a spring to a stream -- LLM operating systems that flow.</i></p>

**ellmos** (XLLM-OS) is a family of text-based operating systems that empower Large Language Models to work autonomously, learn, and self-organize.

## The ellmos Family

All projects follow a **water metaphor** -- growing from a spring to a stream:

| | Project | Description | Repository |
|---|---------|-------------|------------|
| 1 | **[USMC](https://github.com/lukisch/usmc)** | The spring -- shared memory for multi-agent coordination | `pip install usmc` |
| 2 | **[Rinnsal](https://github.com/lukisch/rinnsal)** | The trickle -- USMC + LLM chain orchestration | `pip install rinnsal` |
| 3 | **[BACH](https://github.com/lukisch/bach)** | The stream that unites everything -- full LLM-OS with 109+ handlers, 932+ skills, agents, GUI | `pip install bach` |
